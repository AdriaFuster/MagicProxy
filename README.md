# MTG Proxy

This is a personal project I put together to **automatically download Magic: The Gathering card images in Spanish** and upscale them using a **Hugging Face AI model** to get crisp, high-resolution results.

I built this because finding high-quality scans of MTG cards in Spanish can be a pain, especially for older sets or proxy printing, so I decided to let AI handle the heavy lifting!

---

## Features

* **Spanish Card Downloader:** Automatically fetches card images in Spanish based on card names or set lists.
* **AI-Powered Upscaling:** Integrates with a Hugging Face model to upscale low-res scans into high-quality images.
* **Batch Processing:** I can feed it a list of cards, and it will download and enhance all of them automatically.
* **Clean Output:** Saves the final, crystal-clear upscaled images ready for digital use or printing.
